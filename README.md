# Introduction

<div>
   <h3>Pokédex</h3>

   <p>A web application to create a Pokédex where an API is consumed, allowing you to see the information of each Pokémon.</p>
</div>

## Installation

1. Install dependencies

   ```sh
   npm install
   ```

## Development

2. Start the development server

   ```sh
   npm run dev
   ```

## Production

3. Generate a production build

   ```sh
   npm run build
   ```
